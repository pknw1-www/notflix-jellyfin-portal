02.

Selecting a Jellyfin Client 

Jellyfin is designed to run either via PWA in a browser or by dedicated apps - and because Jellyfin is maintained by the Open Source community rather than a company focused on making money, you can see greater diversity in every feature or function implemented
- community plugins to add features to the server
- crowd sourced metadata analysis 
- fully functional on every platform
- mobile apps etc are free without restriction

While the range of platforms is smaller than Plex, because of the switch to Jellyfin, there will be more and more over the coming months

  • Desktop
  • Mobile
  • Amazon Firestick
  • SmartTV
  • Gaming Devices
  • Other Devices

Jellyfin - much like Plex - is a Web First media server - so most will want to access it on their browser

- Simply opening https://jellyfin.pknw1.co.uk will open the Web Player - login with your username and password
- If you want to run a client on your desktop, check out the Jellyfin Desktop Client on Windows, Mac and Linux
- You can also see all other Desktop clients at the Jellyfin Clients page

For all desktop clients or via the web, you need to follow the same steps
1. either start your browser or the installed desktop client
2. enter the server address either in the address bar in browser, or when prompted for your server address in client
3. you will be connected to the server 
4. when prompted, enter username and password

that's it; the interface is more intuitive than Plex with all the main content right on the home page


The Jellyfin web client will work on all the latest devices and should change the screen for the type of mobile device you have - however I recommend downloading one of the free clients for your device

Android | iOS

I strongly suggest the official Android or iOS client as it's free and updated frequently - at the very least, choose one that doesn't charge you for anything!

Start the App and when prompted, enter the server address then your login details - thats it!


Amazon FireStick devices run Android Apps - so you can select your preferred source and install the App

For LG Smart TV's you can simply install the Jellyfin app from the App Store
